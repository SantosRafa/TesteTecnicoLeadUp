<h2> Coisas Técnicas para execução </h2>
 
  <p> Após clonar o repositorio na sua maquina
  rode o clássico:
  
  yarn ou npm 
  
  em seguida:
  
  yarn start ou npm start
  
  para ativer o jsonServer rode:
  
  node src/server/server.js
  
  </p>
